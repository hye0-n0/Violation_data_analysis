# 학교폭력 가해경험에 관한 영향요인분석: 패널로짓분석과 랜덤포레스트 활용으로
본 연구의 목적은 학교폭력 가해 경험에 영향을 미치는 요인을 파악하는 것이다. 
한국청소년정책연구원에서 수행한 한국아동‧청소년패널조사2018(KCYPS 2018)의 중 1 패널 2, 3, 4차년도 자료를 활용하였다. 패널 로짓분석을 위해 고정효과모형과 임의효과모형 비교 결과, 고정효과모형이 타당하였다. 유의한 변수들을 선택해 랜덤포레스트의 설명변수로 이용하였다.

분석결과는 다음과 같다. 
첫째, 부모 애정도, 부모 강압적 태도, 스마트폰 중독, 가정 소득, 공격성, 우울, 비행, 협동심이 학교폭력에 유의한 영향을 미친다. 
둘째, 랜덤포레스트 변수 별 중요도에서 비행의 중요도가 가장 높았다. 
셋째, 가정 소득은 중요도가 가장 낮게 나타났다. 

이상의 연구결과를 바탕으로 학교폭력 가해경험에 영향을 주는 주요요인에 맞춰 학교폭력 예방책을 마련할 필요가 있다. 학교폭력 근절을 위해 교폭력이 이루어지는 환경을 제거하고, 바람직한 환경을 조성할 수 있는 방안을 모색하였다.
